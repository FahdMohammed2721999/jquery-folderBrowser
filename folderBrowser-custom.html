Folder browser 1.0.0-rc.1

This is a jquery plugin to enable folder browsing with a similar experience to desktop file managers. In this configuration this class will browse folders on a server based on a root that you specify. The server side script which is used to retrieve directory listings in a array could be modified to return various datasets.

This page contains uses a custom icon CSS and the relevant class options defined during class initialisation. This configuration is not dependant on fontAwesome and demonstrates how any iconset can be substituted. The icons used in this demo are from FamFamFam and are high quality despite being a bit small for this example.
